# 🎱 双色球预测自动化系统

基于历史数据统计分析 + 多策略约束引擎，自动生成双色球预测号码，定时获取开奖结果，自动对比中奖等级并通过邮件发送报告。

## 项目目标

- 📊 对 2003 年至今 3400+ 期历史数据进行统计分析，提炼出有效的选号策略
- 🤖 全自动化运行：预测 → 等待开奖 → 对比结果 → 邮件报告 → 生成下期预测 → 循环
- 📧 每期开奖后自动发送 HTML 格式对比报告到邮箱，中奖时标题醒目标注
- 📝 全程日志记录，增量更新数据，状态断点恢复

## 策略引擎

系统综合 5 大核心策略 + 补充约束生成号码：

| 策略 | 规则 | 依据 |
|------|------|------|
| 排除历史重复 | 红球+蓝球组合不在历史中出现 | 3461 期中红球+蓝球从未完全重复 |
| 连号优先 | 至少包含 1 组连号 | 65.1% 的期数含连号 |
| 冷热加权 | 近 50 期频率权重 0.6 + 历史频率权重 0.4 | 热号出现概率更高 |
| 奇偶/和值/跨度 | 奇偶 2:4~4:2，和值 70-130，跨度 15-32 | 合计占 83%+ |
| 上期重号控制 | 与上一期红球重号 0-3 个 | 1-2 个重号占 67.6% |

**补充约束**：三区间 (1-11, 12-22, 23-33) 每区至少 1 个号码

## 中奖判定

| 奖等 | 红球命中 | 蓝球命中 |
|------|---------|---------|
| 一等奖 | 6 | ✓ |
| 二等奖 | 6 | ✗ |
| 三等奖 | 5 | ✓ |
| 四等奖 | 5 或 4 | ✗ 或 ✓ |
| 五等奖 | 4 或 3 | ✗ 或 ✓ |
| 六等奖 | ≤2 | ✓ |

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置邮箱

创建 `.env` 文件：

```env
SMTP_SERVER=smtp.qq.com
SMTP_PORT=465
SENDER_EMAIL=your_email@qq.com
SENDER_AUTH_CODE=your_auth_code
RECEIVER_EMAIL=receiver@example.com
```

### 3. 下载历史数据

```bash
python download_data.py
```

### 4. 启动自动化系统

```bash
python main.py
```

系统启动后会自动：生成 10 组预测 → 定时检查开奖 → 对比结果 → 发送邮件 → 生成下期预测 → 循环运行

## 智能调度

| 时段 | 检查频率 | 说明 |
|------|---------|------|
| 开奖日 (二/四/日) 21:00-23:00 | 每 10 分钟 | 开奖后尽快获取结果 |
| 其他时段 | 每 30 分钟 | 保守检查，防止 API 补数据 |
